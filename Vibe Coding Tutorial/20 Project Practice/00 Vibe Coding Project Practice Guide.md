## I. Why Build Projects?

Before answering "how to build projects," I want to first talk with you about "why build projects."

Many students, after finishing basic tutorials, start frantically learning various tools and technologies, but never build a complete project. The result is: they learn a lot, but feel like they don't know anything.

**Building projects is the only standard for testing learning outcomes.**

More importantly, in the Vibe Coding era, the barrier to building projects has dropped to an unprecedented level. You don't need to master all technical details, you only need to:

1. Know what you want to build
2. Be able to clearly express requirements
3. Know how to use AI tools to implement it

These three points, you've already learned in [Essential Basics]. Now, it's time to apply these abilities to more challenging projects.
