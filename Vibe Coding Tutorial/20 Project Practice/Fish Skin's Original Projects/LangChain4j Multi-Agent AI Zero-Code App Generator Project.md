## Project Introduction

The project has 4 core capabilities.

1) Intelligent Code Generation: Users input requirement descriptions, AI automatically analyzes and selects appropriate generation strategies, generates code files through tool calling, and uses streaming output to let users see AI's execution process in real time.

![](https://pic.yupi.icu/1/1753332332820-9ec614de-65a2-496d-b9b2-dc89c20d06c9.png)



2) Visual Editing: Generated applications are displayed in real time. You can enter edit mode, freely select web page elements and chat with AI to quickly modify pages until satisfied.

![](https://pic.yupi.icu/1/1753332451827-220a1df9-ea60-4646-bea0-64e5f73d15fe.png)



3) One-Click Deployment and Sharing: Generated applications can be deployed to the cloud with one click and automatically capture cover images, getting accessible addresses for sharing, while supporting complete project source code download.

![](https://pic.yupi.icu/1/1753332366033-187b00b1-8609-42b1-ba80-cf58bdb0e970.png)



4) Enterprise Management: Provides backend functions like user management, application management, system monitoring, business metric monitoring, etc. Administrators can set featured applications, monitor AI calls and system performance.
